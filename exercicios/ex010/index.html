Usando links externos

Você pode usar links externos para conectar seu site a outros sites na web. Para criar um link externo, use a tag <a> com o atributo href apontando para a URL desejada.

Usando âncoras para navegação interna

Âncoras permitem que você crie links que levam a diferentes seções da mesma página. Para criar uma âncora, use o atributo id em um elemento e crie um link que aponte para esse ID.
